# Ransomware Ecosystem Network Observatory

Structural analysis of Russia and CIS-linked ransomware networks, mapping the financial, infrastructure, and jurisdictional leverage points across the full criminal ecosystem.

This repository serves the research site at **https://research.therenoproject.org/**.

## What is here

- **Interactive ecosystem map** (current: v2.5): a dependency graph of the ransomware ecosystem showing payment flow, infrastructure dependencies, protection layers, and reconstitution pathways, with phase, jurisdiction, and case-study overlays.
- **Group profiles**: tracked ransomware groups, each backed by live victim data and recent threat-intelligence coverage.
- **Infrastructure profiles**: bulletproof hosting providers and crypto exchanges relevant to the ecosystem.
- **KRYSHA**: an interactive disruption simulator modeling how the ecosystem adapts to financial, infrastructure, legal, and protection-layer pressure.
- **Live intelligence feed**: real-time victim posts sourced from ransomware.live.
- **Document library**: the framework series, fifteen Ecosystem Dependency Project (EDP) module briefs, and monthly activity reports.

## Methodology

The work centers on identifying where the ransomware ecosystem is structurally dependent, and therefore where outside pressure (financial, infrastructure, jurisdictional) can degrade it. Findings distinguish confirmed reporting, credible reporting, and analyst inference, and are organized around measurable indicators rather than narrative alone.

## Data sourcing and automation

Victim data is pulled from the public ransomware.live API. Threat-intelligence context is aggregated from major vendor blogs. These data files refresh automatically on a schedule (see the Actions tab), so the live feed and group pages stay current without manual updates.

## Status and disclaimer

Open research, maintained by Reno. This is an analytical resource. It does not provide operational tooling and contains no malicious code. Reporting is summarized for analysis and may be updated as the ecosystem changes.
